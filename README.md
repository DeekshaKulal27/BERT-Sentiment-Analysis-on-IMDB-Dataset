# 🧠 IMDB Sentiment Analysis using BERT

## 📌 Project Overview

This project focuses on building a sentiment analysis model using a
pre-trained BERT model. The goal is to classify movie reviews from the
IMDB dataset as positive or negative.

## 🎯 Objective

-   Understand BERT for text classification
-   Perform fine-tuning using Hugging Face Transformers
-   Evaluate model performance

## 📊 Dataset

-   IMDB Movie Reviews Dataset
-   Classes: Positive, Negative

## ⚙️ Technologies Used

-   Python
-   PyTorch
-   Transformers
-   Scikit-learn

## 🔄 Pipeline

Raw Data → Preprocessing → Tokenization → Training → Evaluation

## 📈 Metrics

-   Accuracy
-   Precision
-   Recall
-   F1 Score
